# GetACat Frontend

This is the frontend code of [GetACat](). I built this web page as a portfolio project because I wanted to test my frontend skills, mainly focusing on [React](https://react.dev/). This page provides information about cats, such as their lifespan and images. This information comes from the backend, which you can find here 👉 [GetACat_Backend](https://github.com/6gDav/GetACat_Backend).
